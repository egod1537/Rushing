## 📄 Introduction

---

Tilt Maze에 로그라이크 장를 더한 게임입니다. 던전에서 몬스터를 무찌르고 주어진 보석을 획득하고 다음 던전으로 이동할 수 있습니다.

플레이어는 상하좌우로 이동할 수 있으며 앞에 벽이 있을 때까지 이동합니다. 만약 몬스터가 앞에 있는 경우 멈추고 몬스터에게 공격합니다.
